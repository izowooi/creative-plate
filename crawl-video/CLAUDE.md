# crawl-video 프로젝트

## 하네스: Video Downloader Extension

**목표:** Playwright MCP로 사이트를 분석하여 Chrome Extension을 자동 생성한다.

**트리거:** 영상 다운로드 익스텐션, 크롬 익스텐션, 사이트 분석 관련 작업 요청 시
`video-downloader-extension-orchestrator` 스킬을 사용하라. 단순 질문은 직접 응답 가능.
